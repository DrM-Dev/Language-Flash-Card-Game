# Language Flash Card Game 0.1
*A game to help you memorise most common 500 words in 5 languages!*

_________________
This is a flash-cards game built on the database of frequency words 2018 for 5 languages being
[Chinese, Russian, German, French, Spanish]
https://github.com/hermitdave/FrequencyWords/tree/master/content/2018
the database was made by @hermitdave
:)
_________________
to build/modefy this code you need the following packages:
1. Pandas
2. Tkinter
3. Customtkinter
